# PROBLEM REGISTER

**Статус:** SKELETON. Миграция P-01…P-30 из черновика реестра (05_Реестр_проблем.md) выполняется после появления снапшотов D1–D6 — верификация без снапшотов невозможна (К-1).

Схема записи:
| Поле | Описание |
|---|---|
| ID | P-xx |
| Статус | UNVERIFIED / CONFIRMED-DOC / CONFIRMED-FACT / REFUTED / SUPERSEDED |
| Риск | LOW / MED / HIGH / CRITICAL |
| Доказательства | файл snapshot + строки |
| Адресат | 1С / Bitrix / BI / Бизнес |
| Track | A / B |
| Трассировка | ADR / задача подрядчику |
