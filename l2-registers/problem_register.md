# PROBLEM REGISTER

**Статус:** SKELETON. Источник миграции P-01…P-30 — `l2-registers/problem_register.source.md` (frozen, 2026-07-15; ранее ошибочно упоминался как `05_Реестр_проблем.md`). Метки уверенности (🟢/🟡/🟠/🔴) и цитаты «Откуда растут ноги» в источнике основаны на пересказе аналитика и НЕ являются каноном (К-1). Верификация каждой P-xx против снапшотов D1–D6 выполняется брифом B1; до этого все записи — UNVERIFIED (ADR-004).

Схема записи:
| Поле | Описание |
|---|---|
| ID | P-xx |
| Статус | UNVERIFIED / CONFIRMED-DOC / CONFIRMED-FACT / REFUTED / SUPERSEDED |
| Риск | LOW / MED / HIGH / CRITICAL |
| Доказательства | файл snapshot + строки |
| Адресат | 1С / Bitrix / BI / Бизнес |
| Track | A / B |
| Трассировка | ADR / задача подрядчику |
