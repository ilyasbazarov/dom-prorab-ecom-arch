# dom-prorab-ecom-arch

Инженерная операционная система проекта интеграции 1С ↔ Bitrix24 ↔ CMS (Dom Prorab, E-com запуск).

## Уровни доверия артефактов
- **L0 Канон** (`l0-canon/`) — источники истины: Business Canon, Canonical Order Flow, Domain Model, Invariants, Integration Contract. Изменения только через ADR. Пусто до Фазы 4.
- **L1 Решения** (`docs/02_ADR_LOG.md`) — append-only лог архитектурных решений.
- **L2 Регистры** (`l2-registers/`) — производные рабочие регистры: проблемы, вопросы подрядчикам, допущения, открытые вопросы.
- **L3 Внешние объекты** (`l3-external/`) — immutable снапшоты документации подрядчиков. Объект анализа, НЕ источник истины.

## Связанные репозитории
- `dom-propraba-strategy` — Master Knowledge Base (используется по SHA-pinned ссылке, твёрдые секции: [1], [2], [9]).

## Правила доступа агентов
Только SHA-pinned raw curl. Пересказы и выборочные цитаты третьих лиц не являются доказательством (см. Конвенции К-1).
