## 1С-ИНФ-01 — Консолидированный реестр конфигурационных изменений

| **Атрибут** | **Значение**                                                                                                                                                   |
|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия      | **2.0**                                                                                                                                                        |
| Статус      | Готово к реализации (кроме разделов с пометкой ⚠️ TBD)                                                                                                         |
| Контрагент  | List.kg                                                                                                                                                        |
| Назначение  | Единый источник истины по всем изменениям конфигурации 1С:УТ — реквизитам, справочникам, регистрам, константам. Использовать как чеклист при приёмке и деплое. |

## КАК ЧИТАТЬ ЭТОТ ДОКУМЕНТ

- **Источник** — в каком блоке ТЗ описан контекст применения

- **Порядок реализации** — реквизиты из Группы 0 (инфраструктурные) должны быть созданы первыми, до реализации задач которые их используют

- Все поля без явного По умолчанию — не обязательны к заполнению; существующие документы не затрагиваются (нет миграции данных)

- Единожды созданный реквизит не переименовывается без согласования — имя является частью API-контракта
